# Screen（快速筛选）

> 基座能力，零依赖可跑。有数据源 API 时自动拉取定量数据，无 API 时降级为纯 websearch 定性筛选。

## 触发词

"筛选" / "screen" / "筛一下" / "帮我找 {主题} 的股票" / "哪些股票值得看"

## 流程

### Step 0：确定主题和市场

用户给出筛选方向。如果不明确，提供两个预设模板供选择。

### Step 1：websearch 找候选（发散）

用 websearch 搜索符合主题的股票。搜 2-3 组关键词（中英文都试），目标收集 10-20 只候选 ticker。

搜索策略：
- 行业报告、券商推荐、论坛讨论、财经媒体
- 中文搜索 + 英文搜索并行
- 每组关键词取前 5-10 个结果

### Step 2：拉取定量数据（收敛）

对候选 ticker 列表，调数据源拉基础数据：

**数据源优先级**：
1. 已独立安装并授权的 [Longbridge Skill/CLI](https://open.longbridge.com/zh-CN/skill/)：`longbridge quote {TICKER} --format json`
2. [tushare](https://tushare.pro/register)（A 股 .SH/.SZ）：通过 `tools/daily-watch/scripts/fetch_market_data.py`
3. [FMP](https://financialmodelingprep.com/)（全球，可选）：通过 `tools/daily-watch/scripts/fetch_market_data.py`

**拉取字段**：当前价、市值、P/E、P/B、股息率、近 1 年营收增速、近 3 月涨跌幅。

**无数据源 API 时**：跳过此步，Step 3 改为纯定性筛选（从 websearch 结果中提取公开估值信息），所有数字标 `[待验证]`。

### Step 3：按条件过滤 + 排序

用预设模板或用户自定义条件过滤。

### Step 4：Top 5 深度补充

对排名前 5 的公司，各做一轮 websearch：
- 近期催化剂（新产品、订单、政策利好）
- 主要风险（竞争、监管、估值过高）
- 一句话投资逻辑

### Step 5：输出

**格式**：表格 + Top 5 各一段话分析。

**输出路径**：`output/screen/YYYY-MM-DD-{slug}.md`

**模板**：

```markdown
# {主题} 筛选结果

> 日期：YYYY-MM-DD | 模板：{价值股/AI产业链/自定义} | 数据源：{Longbridge Skill/tushare/FMP/websearch}

## 筛选条件

{列出使用的条件}

## 结果总览

| 排名 | Ticker | 名称 | 市场 | 市值 | P/E | P/B | 股息率 | 营收增速 | 近3月涨跌 |
|------|--------|------|------|------|-----|-----|--------|---------|----------|
| 1 | ... | ... | ... | ... | ... | ... | ... | ... | ... |

## Top 5 分析

### 1. {公司名}（{Ticker}）

{一段话：业务概况 + 为什么符合筛选条件 + 近期催化剂 + 主要风险}

...
```

### Step 6（可选）：沉淀

用户看完觉得某只值得跟踪：
- "跟踪这只" → 存到 `wiki/entities/{TICKER}.md`
- "建个假设" → 存到 `hypothesis/H{N}-{slug}.md`

## 预设模板

### 价值股

| 条件 | 阈值 | 说明 |
|------|------|------|
| P/E | < 15 | 低估值 |
| P/B | < 1.5 | 低于净资产 1.5 倍 |
| 股息率 | > 3% | 有现金回报 |
| 市值 | > 10 亿（本币） | 排除微型股 |
| 近 1 年营收增速 | > 0% | 非衰退 |

排序：股息率降序 → P/E 升序

### AI 产业链

| 条件 | 阈值 | 说明 |
|------|------|------|
| 营收增速 | > 20% | 高成长 |
| 市值 | > 5 亿（本币） | 排除壳股 |
| 业务关键词 | AI / 人工智能 / 大模型 / GPU / 算力 / 数据中心 / 云计算 / inference / LLM | 命中至少 1 个 |

排序：营收增速降序 → 市值降序

## 数据标注

所有数字必须标注来源：
- `[Longbridge Skill]`：独立 Agent 扩展；`[tushare]` / `[FMP]`：脚本 API 拉取
- `[网页]`：websearch 公开信息
- `[待验证]`：无 API、数据来自定性描述

## 边界

- 筛选结果是起点，不是投资建议。
- 不自动建仓或下单。
- 无 API key 时仍然可用（降级为定性筛选）。
